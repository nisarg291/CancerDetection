Cancer Detection
Cancer Detection is a website that analyzes the input and detects the possibility of cancer. It also optimizes the code so that the performance will increase. In this website we can see that using convolution network users can detect cancer and according to their budget hospitals would be recommend to them. The benefits of this website are to maximize the resource utilization, good accuracy and save the time.



This system will used in cancer prediction with accurate result. The system will detect the cancer by the use of deep learning neural network. This system will take the user input image and try to predict cancer based on the training dataset. The system has deep learning training model (CNN) which is trained by the training dataset from the sources like GIT, kaggle and reddit. The system also recommends the hospital to the users according their budget and location. It will use the machine learning recommendation algorithm to recommend the hospitals to the users.
